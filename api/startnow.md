# 用JS任意控制手机上的APP

## 1. 控制代码
*  可以用如下简单的JS代码，控制手机上的第三方APP的行为，实现自动测试等目的 *

```
launchApp("微信"); 
click("发现");
click("朋友圈");
sleep(3000);
click(desc("评论"));

```
